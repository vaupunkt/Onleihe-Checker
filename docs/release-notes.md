### 🚀 Installation
1. Extension-Dateien herunterladen
2. Developer-Modus in Chrome aktivieren
3. Unpacked Extension laden
4. Bibliothek im Popup auswählen
5. Amazon.de browsen und automatische Anzeige genießen!
